# 本地部署

## 🚀 快速开始

### 方式一：一键部署（推荐）

```powershell
# 以管理员身份打开 PowerShell，进入项目目录
cd platform-management-main

# 执行一键部署脚本
.\deploy.ps1
```

---

### 方式二：手动分步部署

---

#### 📋 前置条件

| 软件 | 下载地址 |
|------|---------|
| MySQL 8.0+ | https://dev.mysql.com/downloads/mysql/ |
| MongoDB 6.0+ | https://www.mongodb.com/try/download/community |
| Python 3.10+ | https://www.python.org/downloads/ |
| Node.js 18+ | https://nodejs.org/ |

---

#### 1. 启动数据库服务

**MySQL:**
```powershell
# 方式一：服务方式启动
net start MySQL80

# 方式二：手动启动
mysqld --console
```

**MongoDB:**
```powershell
# 1. 创建数据目录（首次）
mkdir C:\data\db -Force

# 2. 启动
mongod
```

---

#### 2. 导入 MongoDB 备份数据

```powershell
cd backend
python -m scripts.import_mongo
```

✅ 自动导入 `MongoData/all_collections_backup.json` 中的所有集合

---

#### 3. 初始化 MySQL 表结构

```powershell
cd backend
python -c "
from app.core.database import engine, Base, ensure_schema_compatibility
import app.models.admin_user, app.models.device, app.models.video
Base.metadata.create_all(bind=engine)
ensure_schema_compatibility()
print('OK')
"
```

✅ SQLAlchemy 自动创建全部 10+ 张数据表

---

#### 4. 启动后端

```powershell
cd backend

# 创建虚拟环境（首次）
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py
```

✅ 后端运行在 **http://localhost:8000**

---

#### 5. 启动前端

```powershell
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

✅ 前端运行在 **http://localhost:3000**

---

## 📦 备份数据说明

`MongoData/all_collections_backup.json` 包含：

| 集合 | 说明 |
|------|------|
| `worker` | 作业人员数据 |
| `alarm_record` | 告警记录（含截图/录像路径） |
| `personnel` | 人员信息 |
| `device` | 定位设备 |
| `video_device` | 摄像头设备 |

## ❓ 常见问题

**Q: MongoDB 连接失败？**
> 确保已执行 `mkdir C:\data\db` 且已运行 `mongod`

**Q: MySQL 密码不对？**
> 修改 `backend/app/core/database.py` 第 18 行的密码配置

**Q: 端口被占用？**
```powershell
# 查看端口占用
netstat -ano | findstr :3000
netstat -ano | findstr :8000
```
