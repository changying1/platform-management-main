# 快速启动指南

---

## 🚀 1 分钟启动 MongoDB

```
双击：启动MongoDB.bat
```

✅ 数据自动存在 `../MongoData/db/` 文件夹里，跟着项目走

---

## 📦 导入备份数据

```powershell
cd backend
python -m scripts.import_mongo
```

✅ 自动检测备份文件位置：
- 优先 `../MongoData/all_collections_backup.json` （外层）
- 后备 `./MongoData/all_collections_backup.json` （内层）

---

## 🗄️ MySQL

你本地已有 MySQL，启动后端时 SQLAlchemy 自动建表：
```
cd backend
venv\Scripts\activate
python main.py
```
